Visualising Network Traffic Information

The following 3 test scenarios are designed to test and record the time taken for a user to identify network traffic issues, and compares the "traditional" dashboard approach with two alternative data visualisations methods.

The object of the test is to record the time taken to find the answer to a question asked about each data set, using only information available on screen.
